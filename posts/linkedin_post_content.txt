🚀 Exciting News! I've Built a Real-Time Loan Approval Prediction System using Machine Learning! 🏦💰

Just finished developing a complete Loan Approval Prediction application that combines the power of XGBoost and HistGradientBoosting classifiers with an intuitive Gradio web interface!

🎯 What This Project Does:
• Predicts loan approval status based on multiple factors including income, home ownership, loan intent, interest rates, and previous defaults
• Features intelligent rule-based pre-processing for instant decisions on straightforward cases
• Provides real-time predictions through an interactive web interface

💡 Key Features:
• Machine Learning Models: XGBoost & HistGradientBoosting classifiers for accurate predictions
• Smart Pre-processing: Custom business rules for quick decision-making on low-risk applications
• Web Interface: Gradio-powered UI for easy interaction
• Production-Ready: Includes serialized model, scaler, and encoder for deployment

🛠️ Tech Stack:
- Python
- XGBoost & Scikit-learn
- Gradio
- Pandas & NumPy
- One-Hot Encoding & MinMax Scaling

📊 The model considers features like:
- Annual income
- Home ownership status (OWN, MORTGAGE, RENT, OTHER)
- Loan intent (Education, Medical, Venture, Home Improvement, Personal, Debt Consolidation)
- Interest rates
- Loan amount
- Previous loan defaults
